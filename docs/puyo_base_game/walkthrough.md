# ぷよぷよ基本ゲーム 実装完了

PythonとPygameを使用したぷよぷよの基本ゲームを実装しました。

## 変更内容

### 環境構築
- `requirements.txt`: Pygameを追加。venv環境での実行を想定。

### コアロジック (`src/core/`)
- **座標系**: Index 0 を最下段、Index 11 を最上段（可視）、Index 12 を隠し列（Spawn地点）とする設計を採用。
- **Puyo**: 基本5色 + おじゃま、壁、空気を定義。
- **Field**: 
  - 重力落下処理（中抜け対応）
  - 連結消去判定（4つ以上で消去）
  - 連鎖処理の基盤
- **Game**:
  - ゲームループ管理（操作 -> 落下 -> 消去 -> 生成）
  - 敗北判定
  - ネクストぷよ生成

### UI / 入力 (`src/ui/`, `src/`)
- **Renderer**: Pygameを使用した描画。Y座標をスクリーン座標へ適切にマッピング（上がIndex 12）。
- **InputHandler**: 矢印キー、Z/Xキー等のマッピング。

### エントリーポイント
- `main.py`: ゲームループの実行。

## 検証結果

### 自動テスト
`tests/test_logic.py` を作成し、以下のロジックが正常に動作することを確認しました。
- 重力落下（空中にあるぷよが地面に着地する）
- 積み重ね（既存のぷよの上に積み重なる）
- 連結消去（4つ繋がると消える）
- 隠し列（Index 12）での連結判定

### 手動確認結果
ユーザーによるGUI操作確認を実施済み。以下の動作が正常であることを確認。
- ぷよの移動、回転、高速落下
- 設置、連鎖消去
